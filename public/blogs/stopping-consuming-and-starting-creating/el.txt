# Εισαγωγή

Κάνω μια υπόσχεση στον εαυτό μου στην αρχή του χρόνου να ξεκινήσω να δημιουργώ περιεχόμενο, και τώρα είναι 15 Μαρτίου και δεν έχω παράγει πολλά.

Αυτό το ιστολόγιο θα χρησιμεύσει ως μια ανεπίσημη ανασκόπηση για μένα σχετικά με τα πράγματα που λειτούργησαν, δεν λειτούργησαν και γενικά τα πράγματα που δεν λειτούργησαν/λειτούργησαν, ώστε να δω τι χρειάζεται να βελτιώσω.

Τους τελευταίους μήνες έχω πειραματιστεί με διάφορες μορφές/στυλ περιεχομένου.

- Περιεχόμενο εργασίας
- Γενικές ενημερώσεις ζωής
- Τάσεις

Λογικά έχω τώρα καλά δεδομένα για το τι λειτουργεί και τι δεν λειτουργεί από τις προβολές και τα λάικ. Τα πράγματα που φαίνεται να λειτουργούν για μένα είναι όταν λέω ιστορίες, όπου εγώ είμαι απλώς ένας χαρακτήρας σε μια ιστορία.

Για παράδειγμα έκανα 3 ή 4 reels μιλώντας στην κάμερα δίνοντας γενικές πληροφορίες, και απέτυχαν.

Αλλά στη συνέχεια έκανα ένα reel όπου ενσάρκωσα μια από τις δικές μου εμπειρίες (μια ιστορία) για συγκέντρωση κεφαλαίων και αυτό φαινόταν να πήγε αρκετά καλά. Αλλά για να είμαι δίκαιος, το να φορέσω μια κακή αμερικάνικη προφορά μπορεί να συνέβαλε σε αυτό.

Το περιεχόμενο που δημιουργώ που έχει ελαφρά ψυχαγωγική αξία φαίνεται να λειτουργεί καλά.

# Κοιτώντας τους συνομηλίκους μου

Έχω ξεκινήσει να παρακολουθώ περισσότερα δείπνα δικτύωσης ιδρυτών σε όλο το Λονδίνο, και έκτοτε έχω γνωρίσει μερικούς συναδέλφους ιδρυτές που δημιουργούν περιεχόμενο. Μιλώντας τους, ένας είχε αποκτήσει 10x περισσότερους ακολούθους από μένα σε μια εβδομάδα κάνοντας καθημερινές αναρτήσεις, από ό,τι κάνω εγώ πότε-πότε σε 3 μήνες.

Ξέρω ότι δεν είναι καλό να συγκρίνεις τον εαυτό σου με άλλους, αλλά νομίζω ότι το να δεις άλλα άτομα που προσπαθούν ένα παρόμοιο πράγμα με εμένα για συμβουλές και όχι για να κριτικάρεις τον εaαυτό σου είναι εντάξει.

Επίσης γνωρίζω έναν άλλο φίλο που έχει ένα πιο επαγγελματικό σετ, έχουν ισχυρή καθημερινή συνέπεια και έχουν συσσωρεύσει πάνω από 50k ακολούθους τους τελευταίους μήνες - όταν ξεκίνησαν πρώτα είχαν περίπου 3k.

Είναι σίγουρα δυνατόν να αυξήσεις ακολούθους, απλώς χρειάζεται να καταλάβω πώς.

Κοιτώντας τους προσπαθώ να καταλάβω τι κάνουν που φαίνεται να λειτουργεί και που χρειάζομαι να κάνω.

Εδώ είναι μερικές παρατηρήσεις:

- Η καθημερινή συνέπεια φαίνεται να είναι ένας τεράστιος παράγοντας, τα άτομα που γνωρίζω ότι πάνε καλά στα κοινωνικά μέσα δημιουργούν περιεχόμενο καθημερινά στην ρουτίνα τους.

- Αφήγηση ιστορίας, οι δημιουργοί που γνωρίζω που μαζεύουν ακολούθους λένε ιστορίες. Ενσαρκώνουν, δεν είναι πάντα φυσικοί άνθρωποι - συνήθως ενσαρκώνουν χαρακτήρες και λένε ιστορίες.

- Η μικρογραφία μετράει, αν τα reels μου δεν έχουν μια ενδιαφέρουσα εξώφυλλο, οι άνθρωποι δεν θα τα παρακολουθήσουν. Τα Reels δεν είναι βιβλία, τα κρίνουν από το εξώφυλλό του.

- Χρονική επένδυση, οι άνθρωποι δεν θα επενδύσουν χρόνο όταν κάνουν κύλιση διαμέσου reels για κακά βίντεο. Μαθαίνω δημιουργία περιεχομένου για πρώτη φορά, οπότε το να κάνω μεγάλα reels είναι κακή κίνηση μέχρι να ξέρω τι λειτουργεί και τι δεν λειτουργεί. Ξεκίνησα με μεγαλύτερα reels 1-2 λεπτών, τώρα φαίνεται ότι τα μικρότερα reels 30 δευτερολέπτων πάνε πολύ καλύτερα.

- Χρόνος δημιουργίας, τα reels παίρνουν μερικές ώρες για να κάνω από γύρισμα, επεξεργασία και ανάρτηση. Δεν ξέρω αν είναι γιατί είμαι πραγματικά αργός, που μπορεί να είναι η περίπτωση, αλλά φαίνεται ότι χρειάζεται να κατανείμω περισσότερο χρόνο κάθε μέρα για να τα φτιάχνω. Γνωρίζω δημιουργούς που έχουν ολόκληρες ομάδες και πρακτορεία πίσω τους. Δεν αποκλείω το πείραμα με αυτό, αλλά τώρα φαίνεται ότι χρειάζεται να περάσω περισσότερο χρόνο κάθε μέρα για να τα δημιουργώ - εκτός αν είμαι διατεθειμένος να πληρώσω για εξουσιοδότηση για να το κάνει.

# Η νέα κατεύθυνση

Τώρα αναφοκουοποιώ και νιώθω ότι χρειάζεται να κάνω ένα θεμελιώδες αλλαγή στον τρόπο που προσεγγίζω το περιεχόμενο. Χρειάζεται να κατανείμω χρόνο στη μέρα για να το κάνω και να το κάνω κάθε μέρα. Χρειάζεται να δημιουργώ ιστορίες, να παίζω χαρακτήρες, να γίνω πιο γρήγορος στην παραγωγή τους και να σταματήσω να περνώ τόσο πολύ χρόνο κάθε μέρα να παρακολουθώ το περιεχόμενο άλλων ανθρώπων.

Το πράγμα που βρίσκω πιο δύσκολο είναι η συνέπεια, όπως και με το μάθημα οποιουδήποτε πράγματος, χρειάζεται να κάνω πρακτική, πρακτική, πρακτική.

Όταν αυτοδίδακτα μάθα κωδικοποίηση, έκανα μια καθημερινή συνήθεια να ξοδεύω τουλάχιστον 4 ώρες την ημέρα για τη δημιουργία περιεχομένου, χρειάζεται να κάνω κάτι σαν αυτό με περιεχόμενο. Χρειάζεται να ξοδέψω τουλάχιστον 2 ώρες κάθε πρωί δημιουργώντας περιεχόμενο προτού κάνω οτιδήποτε άλλο για να βεβαιωθώ ότι το κάνω.

Τώρα θα νοικιάζω ένα γραφείο για 3 μήνες ως δοκιμή αντί να δουλεύω από το σπίτι. Νομίζω ότι αυτό θα είναι μια καλή αλλαγή ζωής. Τώρα θα ξυπνήσω, θα φάω, θα εκπαιδευθώ προς το Λονδίνο, θα φτάσω στο γραφείο και θα δημιουργήσω περιεχόμενο πρώτα απ' όλα. Στη συνέχεια θα κάνω τα κανονικά μου πράγματα για την υπόλοιπη μέρα και δεν χρειάζεται να τερματίσω γρήγορα την εργασία για να κάνω ένα reel ή να κάνω ένα κακό reel επειδή βιάζομαι το τελευταίο πράγμα.

Σκεπτόμενος δυνατά, ίσως θα πρέπει να γράψω το περιεχόμενο στο τρένο προς τα κάτω, να το ηχογραφήσω φρέσκο μόλις φτάσω στη δουλειά και στη συνέχεια να το επεξεργαστώ και να το δημοσιεύσω αμέσως.

Όποτε μαθαίνω νέα πράγματα βρίσκω ότι επικεντρώνομαι σε πολύ απλούς βασικούς στόχους για επίτευξη. Έτσι από τώρα οι απλοί μου στόχοι θα είναι:


- Δημιουργήστε περιεχόμενο πρώτα απ' όλα κάθε μέρα για να βεβαιωθείτε ότι γίνεται και ορίστε ένα πραγματικά γρήγορο διαδικασίας παραγωγής. Στη συνέχεια γίνεται και περισσότερο μια καθημερινή ρουτίνα σαν να βουρτσίζω τα δόντια μου
- Χρειάζεται να λέω ιστορίες, να είμαι ένας χαρακτήρας σε μια ιστορία και να κάνω περιεχόμενο재ά να παρακολουθώ καθώς και ενδιαφέρον.

Μπορεί να πηγαίνω στο λάθος μονοπάτι εδώ, αλλά νιώθω ότι η αλλαγή είναι αυτό που χρειάζομαι αυτή τη στιγμή. Τώρα θα περάσω μερικούς μήνες στη νέα κατεύθυνση και θα δούμε πώς θα πάει. Αλλά νιώθω καλά που ακόμα προσπαθώ ενεργά να κάνω περιεχόμενο και όχι απλώς να σταματήσω μετά από μια εβδομάδα.
