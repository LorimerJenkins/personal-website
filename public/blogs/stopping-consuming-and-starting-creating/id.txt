# Pendahuluan

Saya membuat janji kepada diri sendiri di awal tahun untuk mulai membuat konten, dan sekarang sudah 15 Maret dan saya belum menghasilkan terlalu banyak.

Blog ini akan berfungsi sebagai evaluasi kasar bagi saya tentang hal-hal yang berhasil, tidak berhasil, dan hal-hal umum lainnya yang tidak/berhasil sehingga saya dapat melihat apa yang perlu saya tingkatkan.

Selama beberapa bulan terakhir saya telah bereksperimen dengan berbagai format dan gaya konten.

- Konten pekerjaan
- Pembaruan kehidupan umum
- Tren

Saya rasa sekarang saya memiliki data yang baik tentang apa yang berhasil dan apa yang tidak dari jumlah tampilan dan likes. Hal-hal yang tampaknya berhasil bagi saya adalah ketika saya menceritakan kisah, di mana saya hanyalah seorang karakter dalam sebuah cerita.

Sebagai contoh, saya membuat 3 atau 4 reel berbicara ke kamera memberikan informasi umum, dan semuanya gagal.

Tapi kemudian saya membuat satu reel dengan memerankan salah satu pengalaman saya (sebuah cerita) tentang penggalangan dana dan itu tampak berjalan dengan baik. Tapi jujur, saya mengenakan aksen Amerika yang buruk mungkin berkontribusi pada itu.

Konten yang saya buat yang memiliki nilai hiburan sedikit tampaknya berfungsi dengan baik.

# Melihat sesama saya

Saya telah mulai menghadiri lebih banyak makan malam jaringan pendiri di seluruh London, sejak itu saya telah bertemu beberapa pendiri sesama yang membuat konten. Dari berbicara dengan mereka, salah satu telah mendapatkan 10x lebih banyak pengikut daripada saya dalam seminggu dengan posting harian, dibandingkan dengan apa yang saya lakukan secara tidak teratur dalam 3 bulan.

Saya tahu tidak baik membandingkan diri dengan orang lain, tetapi saya merasa melihat orang lain yang mencoba hal serupa dengan saya untuk tips dan bukan untuk mengkritik diri sendiri adalah baik-baik saja.

Saya juga mengenal teman lain yang memiliki setup lebih profesional, mereka memiliki konsistensi harian yang kuat dan telah mengumpulkan lebih dari 50k pengikut selama beberapa bulan terakhir - ketika saya pertama kali memulai mereka memiliki sekitar 3k.

Tentang saja mungkin untuk menumbuhkan pengikut, saya hanya perlu mencari tahu caranya.

Melihat mereka, saya mencoba mencari tahu apa yang mereka lakukan yang tampaknya berhasil yang perlu saya lakukan.

Berikut adalah beberapa pengamatan saya:

- Konsistensi harian tampaknya menjadi faktor besar, orang-orang yang saya kenal melakukannya dengan baik di media sosial membuat konten setiap hari dalam rutinitas mereka.

- Penceritaan kisah, kreator yang saya kenal mengumpulkan pengikut menceritakan kisah. Mereka berakting, mereka bukan selalu manusia alami - mereka biasanya memerankan karakter dan menceritakan kisah.

- Thumbnail penting, jika reel saya tidak memiliki sampul depan yang menarik, orang tidak akan menontonnya. Reel bukan buku, orang menilainya dari sampulnya.

- Investasi waktu, orang tidak akan menginvestasikan waktu saat menggulir reel untuk video buruk. Saya sekarang belajar pembuatan konten untuk pertama kalinya jadi membuat reel panjang adalah langkah yang buruk sampai saya tahu apa yang berhasil dan apa yang tidak. Saya memulai dengan reel yang lebih panjang 1-2 menit, sekarang tampaknya reel yang lebih pendek 30 detik melakukan jauh lebih baik.

- Waktu pembuatan, reel membutuhkan waktu beberapa jam untuk saya buat dari syuting, pengeditan, dan posting. Saya tidak tahu apakah itu karena saya sangat lambat, yang mungkin benar, tetapi tampaknya saya perlu mengalokasikan lebih banyak waktu dari hari saya untuk membuatnya. Saya mengenal kreator yang memiliki seluruh tim dan agensi di belakang mereka. Saya tidak mengesampingkan kemungkinan bereksperimen dengan itu, tetapi sekarang tampaknya saya perlu menghabiskan lebih banyak waktu setiap hari membuatnya - kecuali saya bersedia membayar untuk mendelegasikannya.

- Berhenti menonton reel, saya benar-benar menonton TikTok/Instagram reel setiap hari. Saya perlu berhenti jika saya ingin menjadi kreator. Ada switch nyata yang perlu saya aktifkan di otak saya untuk berhenti mengonsumsi konten dan sebaliknya mulai membuatnya.

# Arah baru

Saya sekarang memfokuskan kembali dan merasa saya perlu membuat perubahan fundamental dalam cara saya mendekati konten. Saya perlu mengalokasikan waktu dalam sehari untuk melakukan ini dan melakukannya setiap hari. Saya perlu membuat cerita, memainkan karakter, menjadi lebih cepat dalam memproduksinya, dan berhenti menghabiskan begitu banyak waktu setiap hari menonton konten orang lain.

Hal yang paling sulit bagi saya adalah konsistensi, seperti halnya pembelajaran apa pun, saya perlu berlatih berlatih berlatih.

Ketika saya mengajarkan diri sendiri untuk coding saya membuat kebiasaan harian menghabiskan setidaknya 4 jam sehari membuat konten, saya perlu melakukan sesuatu seperti itu dengan konten. Saya perlu menghabiskan setidaknya 2 jam setiap pagi membuat konten sebelum saya melakukan apa pun lagi untuk memastikan itu selesai.

Saya sekarang akan menyewa kantor selama 3 bulan sebagai percobaan daripada bekerja dari rumah. Saya pikir ini akan menjadi perubahan hidup yang baik. Saya sekarang akan bangun, makan, naik kereta ke London, sampai ke kantor dan membuat konten hal pertama. Kemudian lakukan hal pekerjaan biasa saya untuk sisa hari dan saya tidak perlu menyelesaikan pekerjaan dengan cepat untuk membuat reel atau membuat reel buruk dari terburu-buru di menit-menit terakhir.

Berpikir keras mungkin saya harus menulis konten di kereta, merekamnya segar segera setelah saya sampai ke kantor dan kemudian mengedit dan mempostingnya segera.

Setiap kali belajar hal-hal baru saya menemukan fokus pada tujuan inti yang sangat sederhana untuk dicapai. Jadi mulai sekarang tujuan sederhana saya akan:

- Buat konten hal pertama setiap hari untuk memastikan itu selesai dan tentukan proses produksi yang sangat cepat. Kemudian itu selesai dan lebih seperti rutinitas harian seperti menggosok gigi
- Saya perlu menceritakan kisah, menjadi karakter dalam cerita dan membuat konten menyenangkan untuk ditonton serta menarik.

Saya mungkin melangkah ke jalan yang salah di sini, tetapi saya merasa membuat perubahan adalah apa yang saya butuhkan sekarang. Saya sekarang akan menghabiskan beberapa bulan dalam arah baru dan lihat bagaimana hasilnya. Tetapi saya merasa baik bahwa saya masih secara aktif mencoba membuat konten dan tidak hanya berhenti setelah seminggu.

Saya hanya harus terus maju dan tidak berhenti.
